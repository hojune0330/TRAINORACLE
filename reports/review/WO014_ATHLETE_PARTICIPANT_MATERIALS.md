## 먼저 알아둘 것

이 자료는 **합성 자료로 연습하는 비교 시뮬레이션**입니다. 아직 실제 선수를
모집하거나 실제 일지를 처리하는 단계가 아닙니다. 화면과 설명을 미리 읽어 보고
무엇이 헷갈리는지 찾기 위한 연습 자료입니다.

- 화면에 나온 계획은 비교용 후보일 뿐입니다.
- **실제 훈련 계획에는 아무 변화도 생기지 않습니다.**
- 실제 달력, 코치 지시, 알림도 바뀌지 않습니다.
- 이 자료를 읽었다고 동의하거나 참여한 것으로 기록되지 않습니다.

실제 참여를 검토하는 날이 오더라도 별도의 안내와 선택 절차가 필요합니다. 그 전에는
어떤 이름, 실제 일지, 메모, 건강 정보도 이 연습 자료에 입력하지 않습니다.

## 이해 확인과 나의 선택

연습에서는 아래 네 문장을 본인이 직접 설명할 수 있는지 확인합니다. 네 항목 모두
맞아야 하므로 기준은 **4/4**입니다. 하나라도 헷갈리면 설명 화면으로 돌아갑니다.

1. 이것은 실제 실행이 아닌 비교 시뮬레이션입니다.
2. 실제 계획과 달력을 바꾸지 않습니다.
3. 미리 보여 준 구조화 항목만 비교 대상으로 삼습니다.
4. 언제든 거절, 잠시 멈추기, 참여 철회를 선택할 수 있습니다.

### 선택권

- **거절해도 불이익이 없습니다.** 일지, 기능, 기존 스티커와 코치 관계는 그대로입니다.
- **잠시 멈추기**를 누르면 새 선택 처리 중단 상태가 됩니다. 코치가 대신 재개할 수 없습니다.
- **참여 철회**를 누르면 그 시점부터 새 선택 처리가 중단됩니다.
- **삭제 요청**은 접수 상태와 아직 삭제되지 않은 상태를 구분해 보여 줍니다. 확정되지
  않은 삭제를 완료라고 말하지 않습니다.
- 쉬기, 통증이나 부상을 알리기, 참여하지 않기는 모두 벌점의 이유가 아닙니다.

스티커는 일지 작성이나 휴식·상태 알림을 중립적으로 확인하는 표시입니다.
**스티커는 참여 보상이 아닙니다.** 참여 지속, 속도, 거리, 통증을 숨기는 행동에는
스티커나 포인트를 주지 않습니다.

## 어떤 데이터를 누구에게 보여 줄지

비공개인 `나만의 메모`는 내용뿐 아니라 존재 여부와 흔적도 비교하거나 전달하지
않습니다. 이 그림자 연습에서는 원문 훈련 메모 전달 금지이며 실제 전송도 없습니다.
향후 사용자가 수신자와 범위를 직접 고르는 별도 공유 기능은 개인정보 검토 뒤에
다시 결정하며, 이 연습의 참여 동의로 자동 허용되지 않습니다.
아래 선택 화면이 이해되는지만 확인합니다.

전송을 선택하려면 매번 다음을 모두 직접 정해야 합니다.

| 내가 정할 것 | 쉬운 뜻 |
|---|---|
| 받는 사람 | 특정 코치, 보호자, 친구 중 정확히 한 사람 |
| 보낼 항목 | 날짜, 구조화 훈련값처럼 화면에 나열된 항목 |
| 보내는 목적 | 왜 보여 주는지 |
| 유효 기간 | 한 번만 또는 만료 시각 |
| 미리보기 | 보내기 전에 실제 항목과 사람을 다시 확인 |
| 최종 확인 | 받는 사람이나 항목이 바뀌면 다시 확인 |

기본으로 선택된 사람이나 항목은 없습니다. `코치` 같은 역할 이름만으로는 보낼 수
없습니다. **자동 알림은 없습니다.** 현재 앱 안 전달은 승인되지 않았으므로 연습의
모든 전송 결과는 `보류`입니다. 본인이 만든 로컬 백업을 직접 보여 주는 행동은 앱
공유 허가와 별개입니다.

## 좋지 않은 일이 생기면

**좋지 않은 일이 생기면** 비교를 계속하는 것보다 멈추고 사람에게 확인받는 것을
우선합니다. 이 기능은 응급 의료 감시가 아니며 진단도 하지 않습니다.

| 상황 | 화면에서 할 일 |
|---|---|
| 자료가 빠졌거나 검토자가 없음 | 일시 정지하고 확인을 기다림 |
| 안전 검토 중, 불편함·통증·걱정 보고 | 일시 정지하고 비진단적 도움 안내 표시 |
| 실제 계획 변경 시도, 개인정보 노출, 몰래 참여 | 즉시 중단 |
| 내가 그만두기로 선택 | 새 선택 처리 종료, 불이익 없음 |

일시 정지는 시간이 지났거나 데이터가 좋아졌다는 이유로 풀리지 않습니다.
**자동 재개되지 않습니다.** 다시 시작하려면 현재 동의, 같은 자료 버전, 멈춘 이유의
해결, 이해충돌 없는 검토, 책임자 확인이 모두 필요합니다. 중단 또는 철회된 상태는
재개하지 않습니다. 불편함 보고가 다른 사람에게 자동 전송되지도 않습니다.

마지막 비교가 끝나도 `성공`, `안전`, `효과 있음`을 뜻하지 않습니다. 새 처리는
멈추고, 폐기·합성 연습 수정·별도 파일럿 검토 중 하나를 다시 결정해야 합니다.
